# VibeGear2

VibeGear2 is an original, open-source, retro-modern arcade road racer for the web that aims to capture the design DNA of Top Gear 2 on SNES—fast handling, four-race tour progression, meaningful between-race upgrades, weather-aware preparation, and a high “one more race” factor—while remaining legally safer, technically modern, and practical to build from the current VibeRacer codebase and its surrounding tooling.
