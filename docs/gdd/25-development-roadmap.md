# 25. Development roadmap

## Prototype phase

Goal: prove feel, not content.

Deliverables:

- pseudo-3D straight/curve/hill renderer
- one placeholder track
- one car
- one AI opponent
- keyboard controls
- lap timing
- restart flow

## Vertical slice phase

Deliverables:

- one polished four-race tour
- garage, credits, repairs, upgrades
- rain and dry tire difference
- 12-car race field
- one music pack
- local save
- one leaderboard endpoint

## Alpha phase

Deliverables:

- 4 tours
- 16 tracks
- 4 cars
- full UI flow
- economy tuning
- AI archetypes
- ghost system
- content validation tools

## Beta phase

Deliverables:

- full 32-track content
- balance pass
- accessibility pass
- mod loader
- contributor docs
- performance profiling
- save migration tests

## v1.0 phase

Deliverables:

- stable release branch
- docs and mod samples
- CI coverage
- browser compatibility matrix
- launch trailer and screenshots
- contributor starter tasks

## Post-launch roadmap

- challenge events
- extra tours
- mobile touch pass
- split-screen experiment
- community curation tools
