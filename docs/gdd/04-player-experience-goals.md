# 4. Player Experience Goals

## Moment-to-Moment Feel

VibeGear2 should feel:

- Fast before it feels realistic.
- Readable before it feels complex.
- Slippery only when the player understands why.
- Dangerous without being random.
- Upgrade-driven without becoming a grind.
- Retro in presentation, modern in usability.

## Session Length

| Session Type | Target Duration |
|---|---:|
| Single race | 2-4 minutes |
| Retry after failure | Under 10 seconds from results screen |
| One regional tour | 12-20 minutes |
| Garage tuning between races | 30-90 seconds |
| Full casual play session | 20-45 minutes |
| Time trial attempt | 1-5 minutes |
| Daily challenge | 5-10 minutes |

## Difficulty Curve

| Phase | Player Experience |
|---|---|
| Region 1 | Learn steering, boost, passing, and basic upgrades. |
| Regions 2-3 | Introduce weather, denser traffic, sharper curves, repair decisions. |
| Regions 4-5 | Require upgrade planning and better boost timing. |
| Regions 6-7 | Add complex elevation, night/fog visibility, high-speed AI. |
| Region 8 | Test mastery without requiring perfect play. |
| Post-game medals | Optional hard targets for expert players. |

## Arcade vs Simulation Balance

VibeGear2 is **arcade-first**.

| System | Arcade Rule |
|---|---|
| Steering | Responsive, exaggerated, forgiving at low speeds. |
| Tires | Affect grip and weather response, not detailed tire physics. |
| Damage | Clear staged penalties, not realistic mechanical modeling. |
| Gear shifting | Optional skill layer, not mandatory for casual completion. |
| Weather | Noticeable but readable modifiers. |
| AI | Creates drama, but must visibly obey rules. |
| Collisions | Punchy and satisfying, not physically exact. |

## One More Race Hooks

| Hook | Implementation |
|---|---|
| Near qualification | Results show how close the player was to 10th, 6th, 3rd, and 1st. |
| Upgrade affordability | Garage highlights `one more race can afford Level 2 tires`. |
| Rival callouts | A named AI rival taunts or praises the player through short original text. |
| Track mastery | Medals for clean race, no boost, underdog win, and time target. |
| Daily challenge | Fixed car, fixed weather, fixed track seed. |
| Ghost replay | Player can race their best lap or a friend's ghost. |
| Mod discovery | Community tracks can be bookmarked and raced instantly. |
