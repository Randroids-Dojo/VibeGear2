# 19. Controls and Input

## Keyboard Layout (Default)

| Action | Primary | Alternate |
|---|---|---|
| Accelerate | Up Arrow | W |
| Brake / Reverse | Down Arrow | S |
| Steer Left | Left Arrow | A |
| Steer Right | Right Arrow | D |
| Pulse Boost | Space | X |
| Shift Up | E | Right Shift |
| Shift Down | Q | Left Shift |
| Pause | Escape | Enter |

## Gamepad Layout (Default)

| Action | Button |
|---|---|
| Accelerate | RT |
| Brake / Reverse | LT |
| Steer | Left stick / D-pad |
| Pulse Boost | A |
| Shift Up | RB |
| Shift Down | LB |
| Pause | Menu |

## Remapping

- All gameplay actions remappable
- Save bindings to local storage
- Show conflicts immediately
- Reset to defaults option

## Transmission Modes

| Mode | Notes |
|---|---|
| Automatic | Default, slightly less optimal |
| Manual | Faster if used correctly |
| Assisted Manual | Prompts and safeguards |

## Accessibility

| Option | Effect |
|---|---|
| Steering assist | Reduces oversteer |
| Reduced spinouts | Converts spins to speed loss |
| Toggle accelerate | Accessibility option |
| Larger HUD | Improves readability |
| Reduced motion | Less camera shake and effects |

## Mobile (Future)

- Touch steering and throttle zones
- Larger UI targets
- 30 FPS fallback
