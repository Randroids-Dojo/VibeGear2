# 17. Art direction

## Overall visual identity

Keywords:

- kinetic
- glossy
- bright
- weathered roads
- toy-like readability
- exaggerated horizons
- poster-like region identity

## Region art themes

Each tour should have a defined visual package:

- palette
- sky treatment
- road shoulder type
- prop categories
- weather presets
- tunnel material set
- UI accent color

## Car design language

Cars should read as:

- compact, stylized sports coupes
- low polygon / high silhouette
- non-licensed
- readable at small screen scale
- color-customizable with decals optional later

## HUD and menu art

HUD should feel like motorsport game UI, not dashboard simulation. Menus should communicate progression clearly and quickly.

## Iconography

Required icon set:

- cash
- repair
- engine
- gearbox
- dry tire
- wet tire
- nitro
- armor
- visibility
- grip
- tour standings

## Accessibility and readability

- Minimum 4.5:1 contrast for text over translucent panels
- Timer and speed readouts never below 16 px on desktop
- Colorblind-safe damage and weather indicators with shape support
- Motion reduction option

## Asset resolution and export guidance

| Asset type | Base recommendation |
| --- | --- |
| Car sprites | 192×96 source, scaled in-engine |
| Roadside props | 64×64 to 256×256 source |
| UI icons | SVG source, rasterized as needed |
| Sky backdrops | 1920×540 layered panoramas |
| FX sheets | 256×256 or 512×512 |
